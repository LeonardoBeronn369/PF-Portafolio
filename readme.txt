Descripción:
Este portafolio personal ha sido creado con el objetivo de presentar mis proyectos y habilidades de una manera profesional y accesible. La aplicación es responsiva y fácil de navegar, permitiendo a los visitantes explorar mi trabajo y contactarme fácilmente.



Características:
Navegación Intuitiva: Utiliza react-router-dom para una navegación fluida entre diferentes secciones.

Diseño Atractivo: Estilos modernos y responsivos gracias a CSS y componentes de React.

Interactividad Mejorada: Integración de react-icons para iconografía atractiva y sweetalert2 para alertas y notificaciones elegantes.

Formulario de Contacto: Implementado con localStorage para almacenar temporalmente los datos del usuario.



Tecnologías Utilizadas:
HTML5: Estructura y contenido de la aplicación.
CSS3: Estilización y diseño responsivo.
JavaScript ES6+: Lógica y funcionalidad.
React: Construcción de la interfaz de usuario de manera eficiente y modular.



Dependencias:
react-router-dom: Manejo de rutas y navegación dentro de la aplicación.
react-icons: Implementación sencilla de iconos populares en los componentes de React.
sweetalert2: Alertas y modales personalizables para mejorar la interacción del usuario.



¡Gracias por visitar mi portafolio! Espero que disfrutes explorando mi trabajo tanto como yo disfruté creándolo.
